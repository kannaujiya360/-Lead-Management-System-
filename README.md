# Automated Lead Management System (LMS)

** Live https://lead-management-tt06.onrender.com

---

## Project Overview

This LMS provides a unified dashboard for all incoming leads from different marketing channels, enabling real-time lead management, analytics, and reporting. It eliminates manual data entry and improves response efficiency for sales teams.

---

## Key Features

### Website Integration
- Automatically captures leads submitted via the website.
- Stores details like **Name, Email, Phone, Service Interested**.

### Meta Ads Integration
- Imports leads from **Facebook campaigns**.
- **Auto-tags** each lead with campaign and source for tracking.

### Google Ads Integration

- Captures campaign and keyword data for ROI analysis.

### Central Dashboard
- Unified view of all leads (**Website, Meta, Google**) in one place.

### Notifications & Alerts

### Analytics & Reports
- Performance insights and conversion tracking.
- Downloadable **Excel file  reports** for campaigns.

---

## Technical Stack

**Frontend:**  
- React.js 
- Tailwind CSS  
- Recharts & Chart.js for analytics  

**Backend:**  
- Node.js & Express.js  
- MongoDB for lead storage  
- RESTful APIs for fast and secure data flow  

**Libraries & Tools:**  
- Axios, Redux Toolkit, React Hook Form  
- react-chartjs-2, recharts  
- jsPDF, FileSaver, XLSX
